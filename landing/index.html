Friendly Interface

User interface was created to be intuitive, easy and clean. Hope you will love it.

All about the sacks

EOC is all about the sacks. Sacks are the place where you can very easily manage all the stuff that you care about.

Easy Management

You can manage your sacks and cohorts easily. Add friends, change user permissions, work together and have fun!
